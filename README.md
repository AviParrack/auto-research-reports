# Auto-Research Reports

Living research reports generated by a multi-pass agentic engine. Each report is a directory of structured data — question DAG, leaf-level claims with evidence, point-by-point source reviews, and dual-model synthesis — rendered into a browseable site by Astro.

**Live site:** *(deploys to Cloudflare Pages on push)*

## Structure

```
reports/{slug}/      # data — one folder per report
  meta.yaml
  00-intake/
  01-tree/
  02-leaves/{leaf}/
  03-synthesis/
  pass-log.jsonl

site/                # Astro app — reads from reports/, generates static HTML
```

## How it works

A separate Claude Code skill (`auto-research`) runs **passes** over each report folder. Each pass reads current state, writes new artifacts, commits, and pushes — Cloudflare Pages auto-rebuilds.

inspo from Steve Newman, [The Centaur Era](https://secondthoughts.ai/p/the-centaur-era):
